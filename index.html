Babel - the transpiler

https://github.com/babel/awesome-babel

Plugins

babel-plugin-feature-flags

babel-plugin-transform-builtin-extend

enable-feature by using flag
allows to extend buildin types like Array

How to use?

.babelrc

        
          {
            "plugins": [
              "babel-plugin-feature-flags",
              "babel-plugin-transform-builtin-extend"
            ]
          }
